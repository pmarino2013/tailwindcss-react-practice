# Intro App - TailwindCSS

Este proyecto es un ejercicio práctico para aprender a usar TailwindCSS en aplicaciones React. Aquí encontrarás ejemplos de componentes estilizados con clases utilitarias de Tailwind, como botones y barras de navegación responsivas.

## Características

- Configuración de TailwindCSS con Vite y React
- Componentes reutilizables con estilos modernos
- Ejemplo de animaciones y transiciones
- Menú responsivo y barra de navegación

## Instalación

1. Clona el repositorio:
   ```sh
   git clone <url-del-repo>
   ```
2. Instala las dependencias:
   ```sh
   pnpm install
   ```
3. Inicia el servidor de desarrollo:
   ```sh
   pnpm run dev
   ```

## Estructura principal

- `src/components/`: Componentes React personalizados
- `src/App.jsx`: Componente principal
- `index.html`: Archivo base

## ¿Qué es TailwindCSS?

TailwindCSS es un framework de CSS utilitario que permite crear interfaces modernas y responsivas de forma rápida, usando clases predefinidas directamente en el HTML/JSX.
